# 00-Go概述

## 1 为何使用 Go

- 开发效率高(完整的开发工程链tools, test, benchmark, builtin .etc)
- 部署简单(Compile once, run everywhere)
- 良好的native http库及模版引擎(无需任何第三方框架)
- 并发模型

## 2 热点项目

- Docker
- Caddy
- Kubernetes
- CockroachDB

## 3 设计初衷

- 针对其他语言的痛点进行设计。
- 并发编程
- 为大数据、微服务、并发而生的通用语言
- 项目转型首选
