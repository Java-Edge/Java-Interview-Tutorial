# 01-羊毛党灰产链路

能产生利益的地方，就离不开黑灰产。

即使利润很低，黑灰产依然会想办法通过机器，批量操作来规模获利。

假设我是羊毛党，去刷无门槛优惠券获利，需要经过哪些步骤？

1. 搜集优惠信息:新手福利，优惠券，红包...
2. 数量庞大的账号:通过这些账号去批量拿优惠券
3. 自动的拿优惠券:一个账号拿一张优惠券，不可能手动
4. 销赃渠道:批量获取优惠券就是为了获利
